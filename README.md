Site portfolio pessoal da **Dra. Joyce**, graduanda em Odontologia. Apresenta os serviços oferecidos e facilita o contato via WhatsApp e Instagram.

## Stack

Zero frameworks · Zero build tools · Zero dependências

| | |
|---|---|
| HTML5 | Estrutura — SPA com navegação manual via JS |
| CSS3 | Tema rosa gradiente, animações, layout responsivo |
| JavaScript vanilla | Troca de páginas, fade-in por scroll, hamburger menu |
| Google Fonts | Playfair Display + DM Sans |

## Estrutura

```
src/
├── index.html
├── styles/main.css
├── scripts/main.js
└── images/
    ├── logo.png
    └── joyce_1.png
```

## Rodando localmente

```bash
# abre direto no browser
start src/index.html          # Windows
open  src/index.html          # macOS

# ou use a extensão Live Server no VS Code
# clique em "Go Live" com index.html aberto
```

## Deploy

Push na `main` → GitHub Actions faz o deploy automático a partir da pasta `src/`.

Na primeira vez: **Settings → Pages → Source → GitHub Actions**
